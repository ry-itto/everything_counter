# なんでもカウンター
名前の通りなんでもカウントできて保存できるアプリ。
 講義を休んだ回数とかカウントできたらいいなって
 
 ## 仕様
 - カウントするもののリストが見れる
 - カウントするものがカウントできる
 - カウントするものを追加できる
 - カウントするものを隠す/削除できる
